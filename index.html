git和gitee应用
2020-12-10

一、由于国内环境，github访问速度受限。尝试git配合gitee使用；

二、推荐Git学习网站： 访问菜鸟教程!

三、个人使用git配合gitee步骤如下：

1.找到文件夹，右击Git Bash Here

2.输入个人信息
git config --global user.name "FLD"
git config --global user.email FLD@163.com

3.在本地项目目录创建本地仓库
git init

4.上传代码至本地仓库
git add . (add后有. 且与.空格分开)

【知识点】git add . git会递归的将你执行命令时所在的目录中所有文件添加;
git add * git会自动把你当前目录所有修改过的文件添加;

5.提交信息
git commit -m "提交信息"

6.gitee仓库创建，并获取SSH地址

7.管理本地仓库并上传代码
git remote add origin [ssh ID]
git push origin master

先下载再上传
